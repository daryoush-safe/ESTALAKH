8-stage pipeline · OpenCV grid extraction · ResNet-18 orientation · EfficientNet-B1 recognition · inverse-perspective overlay

Stage 01

Input

The photo as uploaded — any rotation, angle, or lighting.

Stage 02

Preprocessing

CLAHE evens out the lighting, median blur kills sensor noise, and an adaptive threshold lifts the ink off the paper — even under shadows.

Stage 03

Grid detection

The largest convex quad wins — with a Hough-line rebuild and a min-area-rect as fallbacks when the border fragments.

Stage 04

Orientation & warp

A perspective warp flattens the grid to a 450×450 bird's-eye view; a ResNet-18 reads its rotation and digit verification breaks the 0°/180° tie.

Stage 05

Line removal

Long morphological kernels erase the grid lines; per-cell connected-component filtering sweeps up the residue, leaving only digits.

Stage 06

Digit recognition

An EfficientNet-B1 classifies all 81 cells. Blue tint marks the model's uncertainty — hover for exact confidence, click a cell and type to correct a misread, then re-solve.

Stage 07

Solver
